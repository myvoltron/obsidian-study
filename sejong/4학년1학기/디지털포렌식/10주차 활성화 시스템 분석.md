# 1. 활성화 시스템 분석의 이해
## 1.1 라이브 포렌식의 중요성
라이브 포렌식은 구동 중인 시스템에서 데이터를 수집하고 분석하는 기술이다. 시스템이 켜져 있는 상태에서만 확인 가능한 정보를 확보할 수 있기 때문에, 침해 사고 현장에서는 중요한 분석 방식이다.
### 1.1.1 개념과 필요성
- 구동 중인 시스템에서 데이터를 수집하고 분석하는 기술이다.
- 사건 현장에서 시스템을 중단하지 않고 실시간 정보를 확보할 수 있다.
- 공격 흔적이나 증거가 훼손되기 전에 확보할 수 있다.
- 디스크 분석만으로 확인하기 어려운 활성 시스템 정보를 확보한다.
### 1.1.2 휘발성 데이터
휘발성 데이터는 전원이 차단되면 소멸되는 임시 데이터이다. 라이브 포렌식에서 가장 먼저 수집해야 하는 핵심 대상이다.
| 구분 | 내용 |
| --- | --- |
| 특성 | 실행 중일 때만 존재하며, 종료·재부팅 시 사라질 수 있음 |
| 주의점 | 수집 중에도 값이 변할 수 있음 |
| 예시 | 실행 중인 프로세스, 네트워크 연결, 사용자 세션, CPU/RAM 사용 현황 |
| 분석 의미 | 사고 당시 시스템 상태와 공격 흔적 복원 |
### 1.1.3 라이브 포렌식 수행 시 주의점
라이브 포렌식은 구동 중인 시스템을 대상으로 하기 때문에, 증거 수집 행위 자체가 시스템 상태를 변경할 수 있다.
- 수집 도구 실행 자체가 메모리, 프로세스, 로그를 변경할 수 있다.
- 시스템 부하를 최소화하고 필요한 명령만 실행해야 한다.
- 수집 순서, 실행 명령, 사용 도구를 기록해야 한다.
## 1.2 라이브 포렌식의 주요 도구 및 기술
라이브 포렌식에서는 메모리 덤프, 메모리 덤프 분석 도구, 시스템 정보 확인 도구, 네트워크 분석 도구 등이 활용된다.
### 1.2.1 메모리 덤프
- **정의**: 구동 중인 시스템의 메모리 내용을 그대로 저장하는 작업이다.
- **목적**: 사건 현장의 사진을 촬영하는 것과 유사하며, 실행 중인 프로세스, 네트워크 연결, 사용자 세션 등의 휘발성 데이터를 보존할 수 있다.
- **수집 도구**: `DumpIt`, `Microsoft Crash Dump`
### 1.2.2 메모리 덤프 분석 도구
- **Volatility**: 메모리 덤프 분석에 사용되는 오픈소스 도구이며, 메모리 덤프에서 다양한 정보를 추출할 수 있다.
- **Rekall**: 구글에서 개발한 오픈소스 도구이며, 메모리 내 시스템 정보와 행위 흔적을 분석하는 데 사용된다.
### 1.2.3 시스템 정보 및 실시간 행위 확인 도구
- **Sysinternals Suite**
    - 실행 프로세스를 확인할 수 있다.
    - 자원 사용 현황을 확인할 수 있다.
    - 네트워크 연결 정보를 확인할 수 있다.
    - 시스템의 실시간 행위 분석에 활용된다.
- **Wireshark**
    - 네트워크 패킷 분석 도구이다.
    - 네트워크 통신을 심층적으로 관찰할 수 있다.
    - 전체적인 네트워크 흐름을 거시적으로 파악할 수 있다.
## 1.3 라이브 포렌식 절차
라이브 포렌식은 시스템 상태 확인에서 시작해 메모리 이미지 획득, 시스템 기본 정보 수집, 비휘발성 데이터 수집으로 이어진다.
### 1.3.1 수집 절차 요약
| 단계 | 수집 대상 | 도구 예시 | 주의점 |
| --- | --- | --- | --- |
| 시스템 상태 확인 | 전원 상태, 로그인 상태 | 육안 확인, 기본 명령 | 켜진 시스템은 즉시 종료하지 않음 |
| 메모리 이미지 획득 | 물리 메모리, 가상 메모리, 시간 민감 데이터 | `DumpIt`, `FTK Imager` | 가장 먼저 수집 |
| 시스템 기본 정보 수집 | 프로세스, 사용자 세션, 네트워크 연결 | `Sysinternals Suite`, `Wireshark` | 명령 실행 기록 보존 |
| 비휘발성 데이터 수집 | 파일 시스템 메타데이터, 로그, 실행 파일 | `FTK Imager` | 휘발성 데이터 수집 후 진행 |

### 1.3.2 핵심 원칙
- 사라지기 쉬운 데이터부터 수집한다.
- 시스템 변경을 최소화한다.
- 수집 대상, 도구, 실행 순서를 명확히 기록한다.

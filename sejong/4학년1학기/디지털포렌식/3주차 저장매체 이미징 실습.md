## 1.2 CLI 기반 디스크 이미징 절차
### 1.2.1 명령어 흐름
| 단계 | 목적 | 명령어 | 결과 / 포렌식 의미 |
| --- | --- | --- | --- |
| 1 | 루트 계정 전환 | `su root` | 디스크 조작 권한 확보 |
| 2 | 디스크 확인 | `fdisk -lu` | `sdb`는 분석 대상, `sdc`는 이미징 저장용 디스크로 식별 |
| 3 | 수집 대상 데이터 생성 | `dd if=/dev/urandom of=/dev/sdb count=2` | 분석용 가상 디스크 준비 |
| 4 | 수사관 디스크 초기화 | `dd if=/dev/zero of=/dev/sdc bs=16K` | 이미징 저장 준비 |
| 5 | 파티션 생성 | `fdisk` | 저장매체를 논리적 단위로 분할 |
| 6 | 파일 시스템 생성 | `mke2fs` | `ext2` 파일 시스템 생성 |
| 7 | 생성 확인 | `fdisk -lu` | 파티션 및 파일 시스템 생성 여부 확인 |
| 8 | 마운트 디렉토리 생성 | `mkdir /mnt/target` | 수사관 디스크 접근 경로 준비 |
| 9 | 디스크 마운트 | `mount /dev/sdb /mnt/target` | 디스크를 디렉토리 트리에 연결 |
| 10 | 원본 해시 추출 | `sha256sum /dev/sdb > /mnt/target/insuspect1.sha` | 원본 무결성 기준값 확보 |
| 11 | 디스크 이미징 | `dd if=/dev/sdb of=/mnt/target/inspect1.img bs=16K` | 수집 대상 디스크를 비트 단위 이미지로 저장 |
| 12 | 이미지 해시 추출 | `sha256sum /mnt/target/inspect1.img > /mnt/target/inspect1.img.sha` | 이미지 검증용 해시값 확보 |
| 13 | 무결성 검증 | `cat` | 원본과 이미지 해시값 일치 여부 확인 |

# 2. FTK Imager를 이용한 이미징 파일 분석
## 2.1 FTK Imager 디스크 이미징
### 2.1.1 도구 개요
- 스토리지 디바이스에 대한 이미지 사본을 생성할 수 있다.
- 수집된 디스크 이미지를 대상으로 내부 분석 기능을 제공한다.
- 실행 중인 시스템의 메모리 추출에도 활용할 수 있다.

### 2.1.2 이미징 절차
| 단계 | 작업 | 의미 |
| --- | --- | --- |
| 1 | `Create Disk Image` 선택 | 디스크 이미지 생성 시작 |
| 2 | `Physical Drive` 선택 | 물리 디스크를 소스로 지정 |
| 3 | USB 드라이브 지정 | 수집 대상 선택 |
| 4 | 이미지 타입 `Raw` 선택 | 디스크 비트 배열을 그대로 저장 |
| 5 | 사건 / 증거 정보 입력 | Case Number, Evidence Number, Description 기록 |
| 6 | 저장 위치 설정 | 이미지 파일 및 로그 저장 경로 지정 |
| 7 | `Start` 실행 | 디스크 이미지와 관련 로그 파일 생성 |

### 2.1.3 이미징 파일 검증
- 생성된 결과 로그 파일에 해시값, 조사자 정보, 사건 번호 등이 기록된다.
- 이미징 작업 완료 메시지를 통해 정상 생성 여부를 확인한다.

## 2.2 dd 명령어 기반 이미징
### 2.2.1 Windows 환경 dd 활용
- Windows 환경용 dd 유틸리티 다운로드: `http://www.chrysocome.net/downloads/dd-0.5.zip`
- CLI 기반으로 물리 디스크 이미징을 수행하고 결과를 확인할 수 있다.

## 2.3 FTK Imager 이미지 분석
### 2.3.1 주요 기능
| 기능 | 확인 내용 |
| --- | --- |
| 삭제 파일 확인 | 삭제된 파일은 `X` 아이콘으로 표시되며 복구 가능 여부를 확인할 수 있다. |
| MAC Time 확인 | 수정, 접근, 생성 날짜와 시간을 확인할 수 있다. |
| 미리보기 | 파일 내부 데이터를 렌더링해 미리보기 형태로 확인한다. |
| Hex 뷰 | 파일을 16진수 형태로 열어 원시 데이터를 분석한다. |

# 3. Autopsy를 이용한 이미징 파일 분석
## 3.1 Autopsy 개요
### 3.1.1 도구 특징
- 오픈소스 기반 디지털 포렌식 도구이다.
- GUI를 기반으로 작동하며 내부적으로 Sleuth Kit 명령어 툴킷을 활용한다.
- 하드디스크, USB, 스마트폰 이미지 등 다양한 저장장치 이미지를 분석할 수 있다.

### 3.1.2 주요 기능
| 기능 | 내용 |
| --- | --- |
| 파일 복구 | 삭제 파일 복구 및 내부 디렉토리 탐색 |
| 타임라인 분석 | 사건의 시간적 흐름 재구성 |
| 자동 분석 | 웹 활동, 브라우저 쿠키, 이메일, 채팅 기록 분석 |
| 썸네일 보기 | 이미지 및 비디오 파일 썸네일 모아보기 |

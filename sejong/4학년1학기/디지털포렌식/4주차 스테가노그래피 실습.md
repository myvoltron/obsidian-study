# 2. OpenStego 실습
## 2.1 실행 화면 이해
### 2.1.1 주요 기능
| 기능 | 설명 |
| --- | --- |
| Hide Data | 캐리어 파일에 비밀 데이터를 숨긴다. |
| Extract Data | 스테가노그래피 파일에서 숨겨진 데이터를 추출한다. |
| Password | 존재 은닉과 내용 암호화를 함께 적용할 수 있다. |
| 파일 단위 은닉 | 텍스트뿐 아니라 문서, 이미지 등 파일 단위 은닉이 가능하다. |

## 2.2 메시지 숨기기
### 2.2.1 Hide Data 절차
| 단계 | 작업 | 결과 |
| --- | --- | --- |
| 1 | 캐리어 역할의 이미지 파일 준비 | 데이터를 숨길 매개체 확보 |
| 2 | 페이로드 역할의 비밀 파일 준비 | 이미지 내부에 숨길 데이터 확보 |
| 3 | `Hide Data` 메뉴에서 두 파일 지정 | 캐리어와 페이로드 연결 |
| 4 | 비밀번호 설정 | 선택 사항이며 보안성 강화 |
| 5 | `Generate` 실행 | 육안으로 거의 구별되지 않는 결과 이미지 생성 |

## 2.3 데이터 추출
### 2.3.1 Extract Data 절차
| 단계 | 작업 | 결과 |
| --- | --- | --- |
| 1 | `Extract Data` 메뉴 선택 | 추출 모드 진입 |
| 2 | 스테가노그래피 이미지 선택 | 숨겨진 데이터가 들어 있는 파일 지정 |
| 3 | 삽입 시 설정한 패스워드 입력 | 인증 및 복호화 |
| 4 | 추출 실행 | 숨겨진 파일을 손상 없이 복원 |

# 3. StegAnalysis
## 3.1 StegAnalysis 개요
### 3.1.1 분석 목적
- 스테가노그래피 기술로 숨겨진 데이터를 찾아내고 식별하는 기술이다.
- 디지털 포렌식 관점에서는 정보를 은닉하는 대상을 추적하는 역할을 한다.
- 통계적 이상점이나 구조적 변조를 통해 숨겨진 데이터 존재 여부를 탐지한다.

### 3.1.2 대표 도구
| 도구 / 범주 | 특징 |
| --- | --- |
| Aletheia | 대표적인 스테가노분석 도구 |
| VSL | 이미지 기반 스테가노그래피 탐지에 활용 |
| 오픈소스 / 상용 도구 | 대량 이미지에서 의심 파일을 빠르게 선별하고 오탐을 줄이는 것이 목표 |
